# Day 8 – Arrays Fundamentals

 

## Problems Solved

1. Largest Element in Array  
2. Missing Number in Array  
3. Second Largest Element  

## Concepts Practiced

- Linear array traversal
- Tracking maximum and second maximum values
- Using mathematical formula (n*(n+1)/2) for missing number
- Handling edge cases (duplicate values, no second largest)
- Time complexity optimization (O(n) solutions)

## Key Learning

Strengthened understanding of basic array manipulation and single-pass logic.  
Avoided unnecessary sorting and optimized solutions for better performance.

Transitioning from number-based problems to structured DSA array problems.
